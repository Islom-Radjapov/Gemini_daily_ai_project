## 🚀 O'rnatish

### 1. Reponi fork qiling yoki klonlang

```bash
git clone https://github.com/Islom-Radjapov/Gemini_daily_ai_project.git
cd Gemini_daily_ai_project
```

### 2. Gemini API kalitini oling

1. [Google AI Studio](https://aistudio.google.com/) ga kiring
2. **"Get API key"** tugmasini bosing
3. **"Create API key"** ni tanlang
4. API kalitni nusxalang va xavfsiz joyda saqlang

### 3. GitHub Secret qo'shing

1. GitHub repongizga kiring
2. **Settings** → **Secrets and variables** → **Actions** bo'limiga o'ting
3. **"New repository secret"** tugmasini bosing
4. **Name:** `GEMINI_API_KEY`
5. **Secret:** API kalitingizni kiriting
6. **"Add secret"** tugmasini bosing

### 4. Workflow ruxsatlarini sozlang

1. GitHub repongizda **Settings** → **Actions** → **General** bo'limiga o'ting
2. **"Workflow permissions"** bo'limida **"Read and write permissions"** ni tanlang
3. **"Save"** tugmasini bosing

### 5. Qo'lda sinab ko'ring

1. Repodagi **Actions** tabiga o'ting
2. **"🤖 Kundalik AI Loyiha Generatori"** workflowni tanlang
3. **"Run workflow"** tugmasini bosing
4. Natijani kuzating! ✨
