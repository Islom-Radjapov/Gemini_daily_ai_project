# Simple Terminal Chatbot

## Project Name and Description

This is a simple command-line interface (CLI) chatbot application written in Python. It's designed to run directly in your terminal, providing basic interactive responses based on user input. The bot uses only standard Python libraries, making it lightweight and easy to run without any external dependencies.
