# Interactive Landmark Map

## Project Description
This project is a single-page web application that displays a simple interactive map. The map features several clickable markers representing different landmarks. When a user clicks on a marker, an information panel appears, showcasing details (name, description, and a representative image) about that specific location. The entire application is self-contained within one HTML file, utilizing inline CSS for styling and inline JavaScript for interactivity.

## How to Run
1.  Save the entire content provided for `index.html` into a new file named `index.html` on your computer.
2.  Open the `index.html` file using any modern web browser (e.g., Chrome, Firefox, Edge, Safari).
3.  The interactive map will be displayed, and you can start clicking on the markers.

## Features
*   **Interactive Map Display:** A visually appealing, styled map area.
*   **Clickable Landmarks:** Multiple markers positioned on the map, representing different points of interest.
*   **Dynamic Information Panel:** When a marker is clicked, a dedicated panel displays the landmark's name, a brief description, and a placeholder image related to the location.
*   **Pure Client-Side:** Entirely built with HTML, CSS, and JavaScript, requiring no backend or external server.
*   **Single File Solution:** All code (HTML, CSS, JS) is embedded within a single `index.html` file for easy deployment and sharing.
*   **Responsive Design:** Basic responsiveness to adapt to different screen sizes.
