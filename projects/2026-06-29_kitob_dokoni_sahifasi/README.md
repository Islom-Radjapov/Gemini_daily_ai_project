# online_book_store_page

## Project Name
Online Book Store Page

## Project Description
This is a single-file, interactive web page for an online book store. It allows users to browse a collection of books, view their details, add desired books to a dynamic shopping cart, and see the total price. The design is modern, clean, and fully responsive. All HTML, CSS, and JavaScript are contained within a single `index.html` file, demonstrating a compact yet functional web application.

## How to Run
1.  Save the content of `index.html` into a file named `index.html` on your computer.
2.  Open the `index.html` file with any modern web browser (e.g., Chrome, Firefox, Edge, Safari).
3.  The page will load, displaying the book collection and providing interactive shopping cart functionality.

## Features
*   **Book Display**: A grid layout showcasing various books with their titles, authors, prices, and cover images.
*   **Add to Cart Functionality**: Users can click an "Add to Cart" button for any book to add it to their shopping cart.
*   **Dynamic Shopping Cart**: A collapsible sidebar displays currently added items, their quantities, and individual prices.
*   **Cart Item Management**: Users can remove items from the cart directly from the cart sidebar.
*   **Automatic Total Calculation**: The cart automatically calculates and displays the running total price of all items.
*   **Item Count Indicator**: A small icon in the header shows the current number of items in the cart.
*   **Checkout Simulation**: A "Checkout" button provides a simple alert message indicating a successful "purchase" and clears the cart.
*   **Responsive Design**: The layout adjusts gracefully for different screen sizes, from desktop to mobile.
*   **Single File Solution**: All code (HTML, CSS, JavaScript) is embedded within one `index.html` file for easy deployment and sharing.
*   **Modern and Clean UI**: Utilizes a professional aesthetic with clear typography and intuitive navigation.
