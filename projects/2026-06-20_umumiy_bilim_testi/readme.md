# umumiy_bilim_testi

## Loyiha nomi: Umumiy Bilim Testi

## Loyiha tavsifi:
Ushbu loyiha interaktiv "Umumiy Bilim Testi" dasturi bo'lib, foydalanuvchilarga turli mavzular bo'yicha bilimlarini sinash imkonini beradi. Dastur bitta HTML faylida HTML, CSS va JavaScript texnologiyalarini birlashtirgan holda yaratilgan bo'lib, zamonaviy va foydalanuvchi uchun qulay interfeysga ega. Foydalanuvchi savollarga javob berib boradi, to'g'ri/noto'g'ri javoblar bo'yicha vizual fikr-mulohaza oladi va yakunda o'zining umumiy natijasini ko'rishi mumkin.

## Qanday ishga tushirish mumkin:
1. `umumiy_bilim_testi.html` faylini kompyuteringizga yuklab oling. Buning uchun pastdagi to'liq kodni nusxalash va uni `.html` kengaytmasi bilan (`umumiy_bilim_testi.html` kabi) saqlash kifoya.
2. Saqlangan `umumiy_bilim_testi.html` faylini istalgan zamonaviy veb-brauzerda (masalan, Google Chrome, Mozilla Firefox, Safari, Microsoft Edge) oching.
3. Dastur avtomatik tarzda ishga tushadi va birinchi savolni ko'rsatadi.

## Xususiyatlar ro'yxati:
*   **12 ta Umumiy Bilim Savollari:** Turli mavzularni qamrab oluvchi keng qamrovli savollar majmuasi.
*   **Interaktiv Javob Berish:** Har bir savol uchun to'rtta variantdan birini tanlash imkoniyati.
*   **Vizual Fikr-mulohaza:** Foydalanuvchi javob berganidan so'ng, to'g'ri yoki noto'g'ri javoblar darhol yashil (to'g'ri) yoki qizil (noto'g'ri) ranglar bilan belgilanadi.
*   **Natijalarni Hisoblash:** Test oxirida foydalanuvchining umumiy to'g'ri javoblar soni ko'rsatiladi.
*   **Testni Qayta Boshlash:** Foydalanuvchilar testni istalgan vaqtda qayta boshlashlari mumkin.
*   **Yagona Fayl:** Barcha HTML strukturasi, CSS stilistikasi va JavaScript logikasi bitta `umumiy_bilim_testi.html` faylida jamlangan.
*   **Zamonaviy Dizayn:** Toza, chiroyli va minimalist foydalanuvchi interfeysi.
*   **Moslashuvchan Dizayn (Responsive Design):** Dastur turli ekran o'lchamlari (mobil telefonlar, planshetlar, noutbuklar) uchun moslashuvchan tarzda ko'rinadi.
