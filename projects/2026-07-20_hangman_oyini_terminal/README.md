# Python Terminal Hangman Game

## Loyiha nomi va tavsifi

"Python Terminal Hangman Game" - bu terminal/konsol muhitida ishlaydigan klassik "Hangman" o'yinining sodda va to'liq Python ilovasi. O'yinchi noma'lum so'zni harfma-harf topishga harakat qiladi. Har bir noto'g'ri taxmin hangman shaklini bir qadamga yaqinlashtiradi. O'yinchi so'zni topishdan oldin to'liq shakl chizilsa, o'yin tugaydi.

Ushbu loyiha faqat Pythonning standart kutubxonalaridan foydalanilgan holda yozilgan bo'lib, hech qanday qo'shimcha paketlarni o'rnatishni talab qilmaydi. Bu kodning soddaligi, tushunarliligi va keng tarqalganligi bilan ajralib turadi.
