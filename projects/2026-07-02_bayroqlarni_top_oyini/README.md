# bayroqlarni_top_oyini

## Loyiha nomi: Bayroqlarni Top O'yini

## Tavsif
Bu interaktiv veb-viktorina dasturi bo'lib, foydalanuvchiga turli mamlakatlarning bayroqlarini ko'rsatadi va qaysi davlatga tegishli ekanligini to'rt xil variant ichidan topishni so'raydi. O'yin yakunida foydalanuvchining umumiy natijasi ko'rsatiladi va o'yinni qayta boshlash imkoniyati beriladi.

## Qanday ishga tushirish mumkin?
Loyiha bitta `index.html` faylidan iborat. Uni ishga tushirish uchun quyidagi qadamlarni bajaring:
1. Yuqoridagi `index.html` fayli tarkibini yangi faylga nusxalang va uni `index.html` deb saqlang.
2. Saqlangan `index.html` faylini istalgan veb-brauzerda (Chrome, Firefox, Edge va hokazo) oching.
3. "Boshlash" tugmasini bosib, o'yinni boshlang!

## Xususiyatlar ro'yxati
*   **Interaktiv o'yin:** Foydalanuvchi tugmalarni bosish orqali javoblarni tanlaydi.
*   **Kamida 10 ta savol:** O'yin har safar 10 ta turli bayroq savollarini taqdim etadi.
*   **Tasodifiy savollar:** Har bir o'yin boshlanganda savollar ro'yxati tasodifiy tartibda aralashtiriladi.
*   **Variantlar aralashmasi:** Har bir savol uchun javob variantlari ham tasodifiy aralashtiriladi.
*   **Natijalarni kuzatish:** To'g'ri javoblar hisoblanadi va o'yin oxirida umumiy natija ko'rsatiladi.
*   **Qayta o'ynash imkoniyati:** O'yin tugaganidan so'ng foydalanuvchi o'yinni qayta boshlashi mumkin.
*   **Vaziyatni ko'rsatish:** Joriy savol raqami ko'rsatiladi (masalan: "Savol 3 / 10").
*   **Vizual fikr-mulohaza:** Javob tanlanganda to'g'ri yoki noto'g'ri ekanligi vizual tarzda ko'rsatiladi.
*   **Zamonaviy dizayn:** HTML, CSS va JavaScript yordamida chiroyli va qulay interfeys yaratilgan.
*   **Yagona fayl:** Barcha HTML, CSS va JavaScript kodlari bitta `index.html` faylida birlashtirilgan bo'lib, oson joylashtirish va ishga tushirish imkonini beradi.
