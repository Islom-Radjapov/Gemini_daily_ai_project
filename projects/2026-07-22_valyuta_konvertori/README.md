# valyuta_konvertori

## Loyiha nomi
Valyuta Konvertori

## Loyiha tavsifi
Bu loyiha HTML, CSS va JavaScript yordamida bitta faylda yaratilgan sodda va zamonaviy valyuta konvertoridir. Foydalanuvchilar tanlangan valyutalar o'rtasida miqdorni tez va oson konvertatsiya qilishlari mumkin. Loyihaning asosiy maqsadi — to'liq ishlaydigan, chiroyli va bitta faylda joylashgan veb-ilovaning namunasini ko'rsatish.

**Eslatma:** Valyuta kurslari faqat demo maqsadida kodga statik ravishda kiritilgan. Real-time, aniq valyuta kurslari uchun tashqi API bilan integratsiya talab qilinadi, bu esa bitta HTML fayl kontekstida (API kalitini ochiq qoldirmasdan yoki CORS cheklovlari tufayli) amalga oshirilishi qiyin.

## Qanday ishga tushirish mumkin
1. Yuqoridagi `index.html` faylining tarkibini o'zingizning kompyuteringizda `index.html` nomli yangi faylga saqlang.
2. Saqlangan `index.html` faylini istalgan veb-brauzerda (Chrome, Firefox, Edge va hokazo) oching.
3. Loyiha darhol ishga tushadi.

## Xususiyatlar ro'yxati
*   **Yagona fayl:** Barcha HTML, CSS va JavaScript kodlari bitta `index.html` faylida joylashgan.
*   **To'liq ishlaydi:** Valyuta konvertatsiyasi funksiyasi to'liq ishlaydi (statik kurslar bilan).
*   **Chiroyli dizayn:** Zamonaviy va intuitiv foydalanuvchi interfeysi (UI) mavjud.
*   **Tanlash imkoniyati:** Foydalanuvchilar konvertatsiya qilish uchun boshlang'ich va yakuniy valyutalarni tanlashlari mumkin.
*   **Miqdor kiritish:** Konvertatsiya qilinadigan miqdorni kiritish maydoni.
*   **Natijani ko'rsatish:** Konvertatsiya qilingan miqdor va joriy ayirboshlash kursi ko'rsatiladi.
*   **Javob beruvchi dizayn:** Turli ekran o'lchamlariga moslashuvchan (responsive) bo'lmasa ham, o'rtacha mobil qurilmalarda ham qulay ko'rinadi.
