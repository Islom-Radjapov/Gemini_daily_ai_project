# Flappy Bird Clone (O'zbek)

Ushbu loyiha HTML5 Canvas yordamida yaratilgan klassik Flappy Bird o'yinining sodda kloni. O'yin bitta HTML faylda to'liq ishlaydi, hech qanday qo'shimcha resurslar yoki kutubxonalar talab qilinmaydi.

## Loyiha nomi va tavsifi

**Loyiha nomi:** Flappy Bird Clone

**Tavsif:** Bu loyiha Flappy Bird mexanikasini takrorlaydigan sodda veb-o'yin. Foydalanuvchi qushni gravitatsiya ta'siridan saqlab, naychalar (pipes) orasidan o'tkazishi kerak. Har bir muvaffaqiyatli o'tish ballni oshiradi. Naycha yoki yerga tegish o'yinni tugatadi.

## Qanday ishga tushirish mumkinligi

1.  Yuqoridagi kod blokida berilgan `index.html` fayl tarkibini bitta `.html` kengaytmali faylga saqlang (masalan, `flappy_bird.html`).
2.  Saqlangan `.html` faylni veb-brauzeringizda (Chrome, Firefox, Edge va boshqalar) oching.
3.  O'yin darhol ishga tushishga tayyor bo'ladi.

## Xususiyatlar ro'yxati

*   **Oddiy o'yin mexanikasi:** Flappy Birdning asosiy qoidalari saqlangan.
*   **Gravitatsiya va sakrash:** Qush avtomatik ravishda pastga tushadi, uni sakrash uchun bosish yoki probel tugmasini bosish kerak.
*   **Harakatlanuvchi naychalar:** Naychalar o'yin davomida doimiy harakatlanadi.
*   **To'qnashuvni aniqlash:** Qush naychalarga yoki pastki/yuqori chegaralarga tekkanda o'yin tugaydi.
*   **Hisobni kuzatish:** Har bir o'tgan naycha uchun ball beriladi va ekranda ko'rsatiladi.
*   **O'yin tugashi ekrani:** O'yin tugaganda natija va qayta boshlash uchun ko'rsatma chiqadi.
*   **Qayta boshlash funksiyasi:** O'yin tugagandan so'ng, istalgan tugmani bosish yoki ekranga bosish orqali o'yinni qayta boshlash mumkin.
*   **Bitta fayl:** Barcha HTML, CSS va JavaScript bitta `index.html` faylda joylashgan.
