# Markdown dan HTML ga Konvertor CLI Dasturi

## Loyiha Nomi
`markdown_to_html_converter_cli`

## Tavsif
Bu loyiha Python dasturlash tilida yozilgan kichik, lekin to'liq ishlaydigan buyruq qatori interfeysi (CLI) dasturidir. U Markdown formatidagi fayllarni o'qib, ularni zamonaviy va chiroyli dizaynga ega HTML fayllariga aylantiradi. Dastur faqat standart Python kutubxonalaridan foydalanadi, qo'shimcha uchinchi tomon kutubxonalarini o'rnatishni talab qilmaydi.
