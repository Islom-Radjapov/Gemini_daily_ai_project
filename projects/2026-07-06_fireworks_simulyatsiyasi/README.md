# interaktiv_salut_simulyatsiyasi

## Loyiha nomi: Interaktiv Salut Simulyatsiyasi

### Tavsif
Bu interaktiv san'at asari bo'lib, veb-brauzerda ajoyib salyut namoyishini taqlid qiladi. Foydalanuvchilar sichqoncha yordamida ekran bo'ylab salyutlarni portlatishlari yoki klaviatura orqali tasodifiy salyutlarni ishga tushirishlari mumkin. Har bir salyut rang-barang zarrachalarga bo'linib, gravitatsiya va havo qarshiligi ta'sirida xiralashib yo'qoladi, bu esa jonli va dinamik vizual effekt yaratadi.

### Qanday ishga tushirish mumkin
1.  **`index.html` faylini saqlang:** Yuqoridagi kod blokida berilgan `index.html` fayli tarkibini shunday nom bilan saqlang.
2.  **Brauzerda oching:** `index.html` faylini istalgan zamonaviy veb-brauzer (Chrome, Firefox, Safari, Edge) orqali oching. Shunchaki faylni brauzer oynasiga tortib tashlashingiz yoki faylga ikki marta bosishingiz mumkin.
3.  **Boshlash:** Loyiha avtomatik ravishda ishga tushadi.

### Xususiyatlar ro'yxati
*   **Interaktiv sichqoncha boshqaruvi:** Sichqonchaning chap tugmasini bosish orqali bosilgan joyda salyut portlashini boshlang. Salyut pastki markazdan bosilgan nuqtaga qarab uchadi.
*   **Klaviatura boshqaruvi:** "Space" (bo'sh joy) yoki "F" tugmasini bosish orqali ekraning pastki qismidagi tasodifiy joydan yuqoridagi tasodifiy nuqtaga qarab uchadigan salyutni ishga tushiring.
*   **Dinamik zarrachalar simulyatsiyasi:** Har bir salyut portlaganda ko'plab kichik, rang-barang zarrachalarga bo'linadi.
*   **Fizikaga asoslangan harakat:** Zarrachalar gravitatsiya va havo qarshiligi (ishqalanish) ta'sirida harakatlanadi va asta-sekin pasayib, xiralashib yo'qoladi.
*   **Chiroyli ranglar:** Har bir salyut uchun tasodifiy jonli ranglar tanlanadi.
*   **Silliq animatsiya:** `requestAnimationFrame` yordamida silliq va optimallashtirilgan animatsiya.
*   **Reaktiv dizayn:** Canvas o'lchami avtomatik ravishda brauzer oynasiga moslashadi.
*   **Bitta fayl:** Barcha HTML, CSS va JavaScript kodlari bitta `index.html` faylida birlashtirilgan bo'lib, uni osonlik bilan ishga tushirish va baham ko'rish mumkin.
