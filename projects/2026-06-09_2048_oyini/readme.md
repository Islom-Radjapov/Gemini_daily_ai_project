# 2048 O'yini

## Loyiha nomi: Canvas 2048 Game

## Tavsif
Bu loyiha HTML5 Canvas yordamida yaratilgan klassik 2048 o'yinining sodda, ammo to'liq ishlaydigan versiyasidir. O'yinning maqsadi bir xil qiymatli plitkalarni birlashtirib, 2048 qiymatiga ega plitkani yaratishdir. O'yin bitta HTML faylida joylashgan bo'lib, barcha zarur stil va logikani (inline) o'z ichiga oladi.

## Qanday ishga tushirish mumkin
1.  `index.html` faylini oching.
2.  Uni har qanday zamonaviy veb-brauzerda (Chrome, Firefox, Safari, Edge) ikki marta bosish orqali oching.
3.  O'yin darhol boshlanadi.

## Xususiyatlar ro'yxati
*   **To'liq ishlaydigan o'yin:** Plitkalarni harakatlantirish, birlashtirish, yangi plitkalar paydo bo'lishi va o'yin tugashini aniqlash.
*   **HTML5 Canvas:** Barcha o'yin vizuallari Canvas elementida chizilgan.
*   **Yagona fayl:** Barcha HTML, CSS va JavaScript bitta `index.html` faylida jamlangan.
*   **Reaktiv boshqaruv:** Klaviatura strelka tugmalari (`ArrowUp`, `ArrowDown`, `ArrowLeft`, `ArrowRight`) yordamida o'yinni boshqarish.
*   **Ball hisoblagich:** Plitkalar birlashganida o'yinchi balni yig'adi.
*   **O'yin tugashi holati:** Agar doska to'lib qolsa va boshqa harakatlar yoki birlashmalar mumkin bo'lmasa, o'yin tugaydi.
*   **O'yinni qayta boshlash:** O'yin tugaganida yoki istalgan vaqtda o'yinni qayta boshlash tugmasi mavjud.
*   **Zamonaviy dizayn:** Tilelar uchun ranglar sxemasi va aniq matn shriftlari bilan toza va estetik ko'rinish.
