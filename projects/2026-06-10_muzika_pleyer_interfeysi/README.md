# Simple Music Player

## Project Name: Simple Music Player

## Tavsif (Description)
This project creates a fully functional and interactive web-based music player interface. It is entirely built using a single HTML file, embedding all CSS for styling and JavaScript for functionality directly within it. The player features basic controls such as play/pause, navigating to the next or previous song, displaying album art, song title, artist, a progress bar to show current playback, and a volume control. The design is modern, clean, and provides a pleasant user experience.

## Qanday ishga tushirish mumkinligi (How to Run)
1.  **Faylni saqlash (Save the file):** Copy the entire content of `music_player.html` and save it as a file named `music_player.html` on your computer.
2.  **Brauzerda ochish (Open in browser):** Navigate to the saved `music_player.html` file using your file explorer and open it with any modern web browser (e.g., Google Chrome, Mozilla Firefox, Microsoft Edge, Apple Safari).
3.  **Ishlatish (Start playing):** The music player interface will load. You can click the "Play" button to start playing the first song and interact with the other controls.

*Note: The audio and image assets are loaded from external public URLs. An internet connection is required for the music player to fetch these assets and function correctly.*

## Xususiyatlar ro'yxati (Features List)
*   **Play/Pause Functionality:** Start and stop music playback with a single click.
*   **Next/Previous Song Navigation:** Skip to the next song or go back to the previous one in the playlist.
*   **Dynamic Song Information:** Displays the current song's title, artist, and album art, which dynamically updates with each track.
*   **Interactive Progress Bar:** Visualizes the current playback position. Users can click on any point of the progress bar to seek to a specific time in the song.
*   **Volume Control:** A slider to easily adjust the playback volume.
*   **Album Art Animation:** The album art visually rotates when a song is playing, and pauses when the song is paused.
*   **Automatic Playback:** Automatically plays the next song once the current one finishes.
*   **Single File Deployment:** All HTML, CSS, and JavaScript are contained within one `music_player.html` file, making it extremely easy to deploy and share.
*   **Modern and Clean Design:** Utilizes CSS gradients, `backdrop-filter` for a frosted glass effect, and smooth transitions for an aesthetically pleasing user interface.
*   **Accessibility:** Buttons include `aria-label` attributes for improved accessibility.
