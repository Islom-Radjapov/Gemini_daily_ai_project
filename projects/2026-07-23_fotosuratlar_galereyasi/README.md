# interactive_photo_gallery

## Loyiha nomi: Interactive Photo Gallery

## Loyiha tavsifi
Ushbu loyiha bitta HTML faylda to'liq ishlaydigan, interaktiv fotosuratlar galereyasi web sahifasidir. U HTML, CSS va JavaScript (barchasi inline) yordamida qurilgan bo'lib, foydalanuvchilarga rasmlar kolleksiyasini ko'rib chiqish, kichik rasmlarni (thumbnails) bosib to'liq hajmdagi rasmlarni modal (lightbox) ko'rinishida ochish va "Oldingi" / "Keyingi" tugmalari yordamida rasmlar orasida harakatlanish imkonini beradi. Dizayn zamonaviy va mobil qurilmalarga moslashtirilgan (responsive).

## Qanday ishga tushirish mumkin?
1.  Yuqorida berilgan `index.html` faylini kompyuteringizga saqlang.
2.  `index.html` faylini istalgan zamonaviy veb-brauzerda (masalan, Chrome, Firefox, Safari, Edge) oching.
3.  Sahifa darhol galereyani ko'rsatadi. Kichik rasmlarni bosib to'liq ko'rinishda ochishingiz mumkin.

## Xususiyatlar ro'yxati
*   **Bir faylga jamlangan loyiha**: Barcha HTML, CSS va JavaScript bitta `index.html` faylida joylashgan.
*   **Interaktiv galereya**: Rasmlar setkasini ko'rsatadi.
*   **Modal (Lightbox) ko'rinishi**: Kichik rasmni bosganda, to'liq hajmdagi rasm overlay oynasida ochiladi.
*   **Navigatsiya tugmalari**: Modal ichida "Oldingi" va "Keyingi" tugmalari yordamida rasmlar orasida harakatlanish.
*   **Yopish imkoniyati**: Modal oynasini "X" tugmasini bosib yoki rasm tashqarisidagi qorong'u fonga bosib yopish mumkin.
*   **Moslashuvchan (Responsive) dizayn**: Sahifa turli xil ekran o'lchamlariga (ish stoli, planshet, mobil) avtomatik tarzda moslashadi.
*   **Zamonaviy dizayn**: Toza va jozibali foydalanuvchi interfeysi.
*   **To'liq ishlaydi**: Hech qanday placeholder yoki TODO qoldirilmagan.
