# Geografiya Viktorinasi

## Loyiha nomi: Geografiya Viktorinasi

## Tavsif
Bu loyiha interaktiv geografiya testi (viktorina) dasturi bo'lib, foydalanuvchiga geografiyaga oid savollarni beradi va ularning javoblarini tekshirib, yakuniy natijani ko'rsatadi. Dastur HTML, CSS va JavaScript yordamida bitta faylda yaratilgan bo'lib, zamonaviy va foydalanuvchi uchun qulay dizaynga ega.

## Qanday ishga tushirish mumkin
1. Yuqoridagi `geografiya_viktorinasi.html` faylining tarkibini kompyuteringizda yangi `.html` fayliga (masalan, `geografiya_viktorinasi.html`) saqlang.
2. Saqlangan `geografiya_viktorinasi.html` faylini istalgan veb-brauzerda (Chrome, Firefox, Safari va hokazo) oching.
3. Loyiha avtomatik ravishda ishga tushadi va siz savollarga javob berishingiz mumkin.

## Xususiyatlar ro'yxati
*   **Interaktiv test:** Foydalanuvchi javoblarni tanlash orqali savollarga interaktiv tarzda javob beradi.
*   **12 ta savol:** Geografiyaga oid kamida 12 ta savoldan iborat.
*   **Ballarni hisoblash:** Har bir to'g'ri javob uchun ball beriladi va yakuniy natija ko'rsatiladi.
*   **Vizual qayta aloqa:** Foydalanuvchi javob tanlaganida to'g'ri yoki noto'g'ri ekanligi ranglar bilan ko'rsatiladi.
*   **"Keyingi" tugmasi:** Har bir savoldan so'ng keyingi savolga o'tish imkoniyati.
*   **Testni qayta boshlash:** Yakuniy natijadan so'ng testni qayta boshlash tugmasi mavjud.
*   **Bir faylli yechim:** Barcha HTML, CSS va JavaScript kodlari bitta `geografiya_viktorinasi.html` faylida joylashgan.
*   **Zamonaviy dizayn:** Foydalanuvchi uchun qulay va estetik jihatdan yoqimli interfeys.
*   **To'liq ishlaydigan:** Hech qanday "placeholder" yoki "TODO" qismlari mavjud emas.
*   **Professional kod:** Kod sifati yuqori, izohlar ingliz tilida yozilgan.
