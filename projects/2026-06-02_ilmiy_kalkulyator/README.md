# ilmiy_kalkulyator (Scientific Calculator)

## Loyiha nomi va tavsifi
Ushbu "Ilmiy Kalkulyator" loyihasi bitta HTML faylda inline CSS va JavaScript yordamida yaratilgan, to'liq ishlaydigan ilmiy kalkulyatordir. Loyiha zamonaviy va professional dizaynga ega bo'lib, foydalanuvchilar uchun intuitiv interfeysni taqdim etadi. U asosiy arifmetik amallardan tortib, murakkab ilmiy funksiyalargacha bo'lgan hisob-kitoblarni bajarishga qodir.

## Qanday ishga tushirish mumkinligi
1.  Yuqoridagi `index.html` fayl tarkibini o'zingizning kompyuteringizda `index.html` nomli yangi faylga saqlang.
2.  Ushbu `index.html` faylni istalgan zamonaviy veb-brauzerda (Chrome, Firefox, Edge, Safari va boshqalar) oching.
3.  Kalkulyator avtomatik tarzda ishga tushadi va foydalanishga tayyor bo'ladi.

## Xususiyatlar ro'yxati
*   **Asosiy arifmetik amallar:** Qo'shish (`+`), ayirish (`-`), ko'paytirish (`x`), bo'lish (`÷`), modul (`%`).
*   **Ilmiy funksiyalar:**
    *   Daraja (`^`): `x^y` ko'rinishida.
    *   Ildiz (`√`): `√x` ko'rinishida.
    *   Trigonometrik funksiyalar: `sin`, `cos`, `tan`.
    *   Logarifmik funksiyalar: `log` (asos 10), `ln` (natural logarifm).
    *   Eksponensial funksiya: `e^x`.
    *   Faktorial (`n!`).
*   **Konstantalar:** `π` (Pi), `e` (Euler soni).
*   **Ekran boshqaruvi:**
    *   `AC` (All Clear): Barcha kiritilgan qiymatlarni va natijani tozalash.
    *   `DEL` (Delete): Oxirgi kiritilgan belgini o'chirish.
    *   `+/-`: Ko'rsatilgan sonning ishorasini o'zgartirish.
*   **Qavslar:** `(` va `)` murakkab ifodalarni kiritish imkoniyati.
*   **Javobni zanjirlash:** Hisoblash natijasidan keyingi amallarni davom ettirish imkoniyati.
*   **Xato nazorati:** Noto'g'ri ifodalar yoki bo'lish amallaridagi xatoliklarni aniqlash va "Error" xabarini ko'rsatish.
*   **Zamonaviy va responsiv dizayn:** Bitta HTML fayl ichida joylashgan chiroyli va qulay foydalanuvchi interfeysi.
