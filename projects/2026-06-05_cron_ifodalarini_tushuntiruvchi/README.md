# cron_explainer

A simple Python utility that parses and explains standard 5-field cron expressions in human-readable language.

## Project Description

Cron expressions are powerful but can be cryptic. This utility helps developers and system administrators understand what a cron job is scheduled to do by breaking down its expression into individual field explanations and providing a summary. It adheres to the standard 5-field cron format (minute, hour, day of month, month, day of week).

1.  **Save the files**: Ensure you have `main.py` in your project root, and a `cron_explainer` directory containing `__init__.py` and `cron_parser.py`.
