# Savdo Dashboard

Bu kichik, ammo to'liq ishlaydigan va chiroyli dizayndagi savdo ma'lumotlar paneli loyihasi. Loyiha bitta HTML faylidan iborat bo'lib, barcha CSS va JavaScript kodlari inline tarzda yozilgan. U soxta ma'lumotlar bilan ishlaydi va savdo ko'rsatkichlarini, trendlarni, eng ko'p sotilgan mahsulotlarni va so'nggi tranzaksiyalarni ko'rsatadi.

## Qanday ishga tushirish mumkin?

1.  Ushbu loyiha bitta `sales_dashboard.html` faylidan iborat.
2.  Faylni yuklab oling yoki tarkibini `sales_dashboard.html` nomli yangi faylga nusxalang.
3.  `sales_dashboard.html` faylini veb-brauzeringizda (masalan, Chrome, Firefox, Safari) oching.
4.  Dashboard darhol ishga tushadi va ma'lumotlarni ko'rsatadi.

## Xususiyatlar

*   **Chiroyli va Zamonaviy Dizayn:** Toza va professional foydalanuvchi interfeysi.
*   **Bitta Fayl:** Barcha HTML, CSS va JavaScript yagona `sales_dashboard.html` faylida jamlangan.
*   **Soxta Ma'lumotlar:** Haqiqiy ma'lumotlarga o'xshash generatsiya qilingan ma'lumotlar bilan ishlaydi.
*   **Asosiy Ko'rsatkichlar:** Jami Savdo, Yangi Buyurtmalar, O'rtacha Buyurtma Qiymati, Konvertatsiya Darajasi kabi muhim ko'rsatkichlarni ko'rsatadi.
*   **Savdo Trendi:** Oxirgi 7 kunlik savdo trendini vizual bar grafik yordamida ko'rsatadi.
*   **Eng Ko'p Sotilgan Mahsulotlar:** Eng yaxshi sotilgan mahsulotlar ro'yxati.
*   **So'nggi Tranzaksiyalar:** Yaqindagi savdo tranzaksiyalarining qisqa jadvali.
*   **Hududlar Bo'yicha Savdo:** Hududlar bo'yicha savdo hajmini ko'rsatadi.
