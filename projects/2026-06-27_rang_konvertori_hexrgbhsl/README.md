# Color Converter (HEX/RGB/HSL)

## Project Name
Color Converter (HEX/RGB/HSL)

## Description
A simple, single-page web application designed to convert colors between three popular formats: HEX (Hexadecimal), RGB (Red, Green, Blue), and HSL (Hue, Saturation, Lightness). This tool allows users to input a color in any of these formats and instantly see its equivalent representation in the other two formats, along with a live visual preview of the color. The entire application is self-contained within a single HTML file, featuring inline CSS for styling and inline JavaScript for all conversion logic and interactivity.

## How to Run
To run this project, simply follow these steps:
1.  **Save the file:** Copy the entire content provided for `index.html` and save it to a file named `index.html` on your computer.
2.  **Open in browser:** Locate the `index.html` file you just saved and open it with any modern web browser (e.g., Chrome, Firefox, Safari, Edge).

The application will load instantly, ready for use.

## Features
*   **Real-time Conversion:** Instantly converts colors between HEX, RGB, and HSL as you type.
*   **Live Color Preview:** A dedicated display box shows the current color, updating immediately with each valid input.
*   **Bidirectional Conversion:** Input can be provided in any of the three formats (HEX, RGB, or HSL), and the other two will be updated accordingly.
*   **Single File Deployment:** All necessary code (HTML structure, CSS styles, and JavaScript logic) is embedded within a single `index.html` file, making it extremely easy to deploy and share.
*   **Clean and Modern UI:** A user-friendly and aesthetically pleasing interface ensures a smooth user experience.
*   **Robust Input Parsing:** Handles various input formats (e.g., `#RRGGBB`, `RRGGBB`, `rgb(R, G, B)`, `R, G, B`, `hsl(H, S%, L%)`, `H, S, L`) and validates inputs to prevent errors. Invalid inputs do not disrupt the converter but retain the last valid state.
