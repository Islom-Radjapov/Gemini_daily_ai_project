# css_glitch_text_effect

## Loyiha nomi
CSS Glitch Text Effect

## Tavsif
Bu loyiha sof CSS animatsiyalari yordamida yaratilgan, matn uchun raqamli glitch effektini namoyish etuvchi kichik, zamonaviy va to'liq ishlaydigan veb-sahifadir. U minimalist dizayn bilan ajralib turadi va har qanday HTML loyihasiga osongina qo'shish uchun optimallashtirilgan. Matn "GLITCH EFFECT" bo'lib, rang-barang, dinamik va xuddi eski kompyuter ekranidagi kabi ta'sirchan ko'rinish beruvchi effektga ega. Loyiha nol JavaScript ishlatadi.

## Qanday ishga tushirish mumkin
1.  Yuqoridagi `index.html` kodini kompyuteringizda `index.html` nomli faylga saqlang.
2.  `index.html` faylini istalgan zamonaviy veb-brauzerda (Chrome, Firefox, Edge, Safari va boshqalar) oching.
3.  Sahifa avtomatik ravishda glitch effekti bilan matnni ko'rsatadi.

## Xususiyatlar ro'yxati
*   **Sof CSS animatsiyalari:** Barcha effektlar JavaScript-dan foydalanmasdan, faqat CSS-dan foydalangan holda yaratilgan.
*   **Interaktiv dizayn:** Matn raqamli glitch effektini taqlid qiluvchi doimiy ravishda animatsiyalanadi.
*   **Foydalanish qulayligi:** Faqat bitta HTML fayl, ichiga joylashtirilgan CSS bilan, bu integratsiyani juda oson qiladi.
*   **Zamonaviy estetik:** Qora fon, neon ranglar va pikselli shrift (Press Start 2P) retro-futuristik va jozibali ko'rinishni ta'minlaydi.
*   **Moslashuvchanlik:** Matn shrift hajmi viewport kengligiga qarab moslashuvchan bo'lib, turli ekran o'lchamlarida yaxshi ko'rinadi.
*   **Pseudo-elementlardan foydalanish:** HTML strukturani toza saqlash uchun `::before` va `::after` pseudo-elementlari ishlatilgan.
