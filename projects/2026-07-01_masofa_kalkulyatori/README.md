# distance_calculator

## Loyiha nomi: Masofa Konvertori

## Loyiha tavsifi:
Bu oddiy, veb-asosidagi masofa konvertori bo'lib, turli xil masofa birliklari (kilometr, milya, metr, yard, fut, dyuym) o'rtasida o'zgarish imkoniyatini beradi. U bitta HTML faylda joylashgan bo'lib, CSS va JavaScript to'g'ridan-to'g'ri HTML ichiga yozilgan. Dizayn zamonaviy va foydalanuvchiga qulay.

## Qanday ishga tushirish mumkin:
Loyiha faqat bitta fayldan iborat bo'lgani sababli, uni ishga tushirish juda oddiy:
1.  `distance_calculator.html` faylini yuklab oling yoki yuqoridagi kodni nusxalash orqali o'zingizning `.html` faylingizni yarating.
2.  Faylni veb-brauzerda (Chrome, Firefox, Safari va h.k.) oching.
3.  Brauzer avtomatik ravishda kalkulyatorni ko'rsatadi va siz undan foydalanishni boshlashingiz mumkin.

## Xususiyatlar ro'yxati:
*   **Ko'p birlikni qo'llab-quvvatlash**: Kilometr (km), Milya (mile), Metr (m), Yard (yard), Fut (foot), Dyuym (inch) birliklari o'rtasida konvertatsiya.
*   **Foydalanuvchiga qulay interfeys**: Tushunarli kiritish maydoni, birlik tanlash uchun ochiluvchi ro'yxatlar va konvertatsiya tugmasi.
*   **Yagona fayl arxitekturasi**: Barcha HTML, CSS va JavaScript bitta `distance_calculator.html` faylida birlashtirilgan.
*   **Tezkor va samarali**: Boshqa resurslarga bog'liqliksiz tez yuklanadi va ishlaydi.
*   **Xatoni boshqarish**: Noto'g'ri son kiritilgan holatlarda foydalanuvchini ogohlantiradi.
*   **Professional dizayn**: Toza, zamonaviy va ko'zni quvontiradigan interfeys.
