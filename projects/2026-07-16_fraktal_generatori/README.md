# Fraktal Generatori

## Loyiha Tavsifi
Bu loyiha HTML Canvas yordamida Mandelbrot fraktalini interaktiv tarzda vizualizatsiya qiluvchi kichik, ammo to'liq ishlaydigan veb-sahifadir. Foydalanuvchilar fraktalning markazini o'zgartirishi, kattalashtirishi va maksimal iteratsiya sonini sozlashlari mumkin. Loyiha bitta HTML faylda joylashgan bo'lib, barcha CSS va JavaScript kodlari uning ichida yozilgan.

## Qanday Ishga Tushirish Mumkin
1.  `fractal_generator.html` faylini kompyuteringizga yuklab oling.
2.  Faylni o'z veb-brauzeringizda (Chrome, Firefox, Edge va hokazo) oching.
3.  Sahifa yuklangandan so'ng, fraktal avtomatik ravishda chiziladi.

## Xususiyatlar Ro'yxati
*   **Mandelbrot Fraktali:** Matematik jihatdan qiziqarli Mandelbrot fraktalini yuqori aniqlikda chizadi.
*   **Interaktiv Zoom va Pan:**
    *   Fraktalda **ikki marta chap tugmani bosish** (double-click) shu nuqtaga kattalashtiradi.
    *   Fraktalda **o'ng tugmani bosish** (right-click) shu nuqtadan kichiklashtiradi (kontekst menyusini ko'rsatmaydi).
    *   Canvasda **bir marta chap tugmani bosish** (single-click) fraktalning markazini bosilgan nuqtaga ko'chiradi.
*   **Sozlanishi Iteratsiyalar:** Maksimal iteratsiya sonini sozlash orqali fraktalning tafsilot darajasini va ranglarini o'zgartirish mumkin.
*   **Markaz va Zoomni Boshqarish:** Fraktalning markaz koordinatalarini (real va xayoliy qism) va umumiy kattalashtirish darajasini to'g'ridan-to'g'ri kiritish orqali boshqarish.
*   **Qayta O'rnatish:** Fraktalni boshlang'ich holatiga qaytarish tugmasi.
*   **To'liq Bir Faylda:** Barcha HTML, CSS va JavaScript kodlari bitta `fractal_generator.html` faylida jamlangan.
*   **Zamonaviy Dizayn:** Toza, zamonaviy va foydalanuvchi uchun qulay interfeys (Dark Mode).
