# temperature_converter

## Loyiha nomi: Temperature Converter (Harorat Konvertori)

### Loyiha tavsifi
Ushbu loyiha - bu oddiy, ammo to'liq ishlaydigan harorat konvertori bo'lib, u Celsius, Fahrenheit va Kelvin o'rtasida qiymatlarni o'zgartirish imkonini beradi. Loyiha bitta HTML faylidan iborat bo'lib, barcha CSS va JavaScript kodlari shu faylning ichiga (inline) joylashtirilgan. Bu uni yuklab olish va darhol ishlatish uchun juda qulay qiladi. Dizayn zamonaviy va foydalanuvchilar uchun qulaydir.

### Qanday ishga tushirish mumkin
1.  **Faylni saqlash:** Yuqoridagi `temperature_converter.html` fayl tarkibini kompyuteringizga `temperature_converter.html` nomi bilan saqlang.
2.  **Brauzerda ochish:** Saqlangan `temperature_converter.html` faylini istalgan veb-brauzerda (Google Chrome, Mozilla Firefox, Microsoft Edge va boshqalar) ikki marta bosish orqali oching.

Shu bilan loyiha darhol ishga tushadi va siz haroratlarni o'zgartira olasiz.

### Xususiyatlar ro'yxati
*   **Intuitiv interfeys:** Toza va sodda dizayn foydalanishni osonlashtiradi.
*   **Uchta asosiy birlik:** Celsius, Fahrenheit va Kelvin o'rtasida konvertatsiya.
*   **Bir faylli yechim:** Barcha kodlar (HTML, CSS, JavaScript) bitta `temperature_converter.html` faylida.
*   **Darhol ishga tushirish:** Hech qanday qo'shimcha o'rnatishlar yoki konfiguratsiyalar talab qilinmaydi.
*   **Xatolarni boshqarish:** Noto'g'ri kiritilgan qiymatlar uchun xabar berish.
*   **Mobilga mos:** Foydalanuvchi tajribasini yaxshilash uchun responsiv dizayn (kichikroq ekranlarda ham yaxshi ko'rinadi).
