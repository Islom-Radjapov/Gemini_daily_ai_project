## Universal Countdown Timer

This is a small, fully functional, and interactive web-based countdown timer. It allows users to set a specific future date and time, and then displays a real-time countdown to that moment. The timer state is cleverly persisted using local storage, ensuring that your countdown continues even if you close and reopen your browser.

### How to Run
1.  **Save the code:** Copy the entire content of `index.html` into a new file named `index.html`.
2.  **Open in browser:** Open the `index.html` file using any modern web browser (e.g., Chrome, Firefox, Safari, Edge).

### Features
*   **Custom Target Date & Time:** Easily set your desired future date and time using intuitive input fields.
*   **Real-time Countdown:** Displays remaining time in Days, Hours, Minutes, and Seconds.
*   **Persistence:** The countdown state is automatically saved to your browser's local storage. This means if you refresh the page or close your browser, the timer will resume exactly where it left off.
*   **"Time's Up!" Notification:** A clear message appears when the countdown reaches zero.
*   **Reset Functionality:** A dedicated button to stop the current countdown and reset all fields.
*   **User-Friendly Design:** A clean, modern, and responsive interface built with basic HTML, inline CSS, and JavaScript.
*   **Input Validation:** Ensures that only future dates/times can be set.
