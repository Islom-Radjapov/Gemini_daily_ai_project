# Tetris Clone

## Project Description
This is a simple Tetris clone built using HTML5 Canvas and JavaScript. The goal was to create a fully functional, small-scale game that runs entirely within a single HTML file, showcasing basic game development principles like game loops, collision detection, and user input handling.

## How to Run
1.  Save the content of `tetris_clone.html` into a file named `tetris_clone.html` on your computer.
2.  Open the `tetris_clone.html` file with any modern web browser (Google Chrome, Mozilla Firefox, Microsoft Edge, Safari, etc.).
3.  The game will load automatically. Press the "Start Game" button to begin playing.

## Features
*   **Classic Tetris Gameplay:** Move, rotate, and drop tetrominoes to clear lines.
*   **HTML5 Canvas:** All graphics are rendered using the HTML5 Canvas API.
*   **Single File Implementation:** All HTML, CSS, and JavaScript are contained within a single `tetris_clone.html` file for easy portability and sharing.
*   **Responsive Controls:**
    *   **Left Arrow:** Move piece left
    *   **Right Arrow:** Move piece right
    *   **Down Arrow:** Soft drop (move piece down faster)
    *   **Up Arrow:** Rotate piece
    *   **Spacebar:** Hard drop (instantly drop piece to the bottom)
*   **Score Tracking:** Points are awarded for clearing lines.
*   **Game Over Condition:** The game ends when a new piece cannot be placed.
*   **Start/Reset Functionality:** A button to start a new game or restart a finished one.
