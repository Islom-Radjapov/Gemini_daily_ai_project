# interactive_bar_chart_project

## Project Name: Interactive Bar Chart

## Description:
This project demonstrates a simple yet interactive bar chart visualization implemented entirely within a single HTML file. It uses SVG (Scalable Vector Graphics) to render the chart, making it responsive and clear on various screen sizes. The chart visualizes sales data for different months, allowing users to hover over bars to see specific values. This setup ensures ease of deployment as it requires no external files or dependencies.

## How to Run:
1.  Save the provided content as `interactive_bar_chart.html` in a local directory on your computer.
2.  Open the `interactive_bar_chart.html` file using any modern web browser (e.g., Google Chrome, Mozilla Firefox, Safari, Microsoft Edge).
3.  The interactive bar chart will be displayed on the page immediately.

## Features:
*   **Single File Deployment**: All HTML, CSS, and JavaScript are contained within a single `interactive_bar_chart.html` file, simplifying deployment and sharing.
*   **SVG Based Rendering**: Utilizes SVG for sharp, scalable, and responsive graphics that look great on any device or screen resolution.
*   **Interactive Tooltip**: Hovering over any bar on the chart displays a dynamic tooltip showing the bar's category (month) and its exact value (sales amount).
*   **Dynamic Scaling**: The chart automatically scales its bars and axes based on the provided dataset, ensuring accurate representation regardless of value ranges.
*   **Clear Axes and Labels**: Provides well-defined X and Y axes with appropriate labels for easy interpretation of the data.
*   **Modern and Clean Design**: Features a clean, modern aesthetic with a professional color palette and subtle visual effects for a pleasant user experience.
*   **No External Dependencies**: Built using vanilla JavaScript, HTML, and CSS without relying on any external libraries or frameworks.
