# Scientific Calculator (Ilmiy Kalkulyator)

## Project Name & Description
This project is a fully functional scientific calculator implemented in a single HTML file. It boasts a clean, modern, and professional user interface with essential scientific functions alongside basic arithmetic operations. All styling (CSS) and interactivity (JavaScript) are embedded directly within the HTML, making it a highly portable and self-contained application.

## How to Run
To run this scientific calculator, simply save the `scientific_calculator.html` file to your computer and open it with any modern web browser (Google Chrome, Mozilla Firefox, Microsoft Edge, Safari, etc.). No special server or installation is required.

## Features List
*   **Basic Arithmetic Operations:** Addition (+), Subtraction (-), Multiplication (*), Division (/).
*   **Scientific Functions:**
    *   Square Root (√)
    *   Power (x<sup>y</sup>)
    *   Logarithm Base 10 (log)
    *   Natural Logarithm (ln)
    *   Sine (sin - assumes degrees)
    *   Cosine (cos - assumes degrees)
    *   Tangent (tan - assumes degrees)
    *   Pi (π) constant
    *   Euler's number (e) constant
*   **Utility Functions:**
    *   All Clear (AC): Clears all calculations and resets the calculator.
    *   Clear Entry (C): Clears the current input, allowing correction without resetting the entire calculation.
    *   Percentage (%): Converts the current number to a percentage (divides by 100).
    *   Negate (+/-): Toggles the sign of the current number.
    *   Decimal Point (.)
*   **Responsive Design:** The calculator's layout adjusts gracefully to different screen sizes.
*   **Modern UI:** Clean, professional aesthetics with hover effects on buttons.
*   **Single File Implementation:** Everything is contained within `scientific_calculator.html` (HTML, CSS, JavaScript).
