# World Population Statistics Visualization

## Loyiha nomi: Dunyo aholisi statistikasi vizualizatsiyasi

## Tavsif
Ushbu loyiha bitta sahifali HTML ilovasi bo'lib, dunyoning eng ko'p aholili davlatlari statistikalarini HTML Canvas yordamida interaktiv bar diagramma ko'rinishida vizualizatsiya qiladi. Foydalanuvchilar ma'lum bir yilni tanlab, o'sha yil uchun mamlakatlarning aholi sonini ko'rishlari mumkin.

## Qanday ishga tushirish mumkin?
1.  Yuqoridagi kod bloklarini bitta `index.html` fayliga saqlang.
2.  `index.html` faylini istalgan zamonaviy veb-brauzerda (Chrome, Firefox, Edge va hokazo) oching.
3.  Sahifa yuklangandan so'ng, siz dunyo aholisi statistikasi diagrammasini ko'rasiz.

## Xususiyatlar ro'yxati
*   **Interaktiv Bar Diagramma**: Eng ko'p aholili davlatlarning aholi sonini chiroyli bar diagramma ko'rinishida ko'rsatadi.
*   **Yil Tanlash**: Yuqoridagi "Select Year" (Yil tanlang) menyusidan turli yillarni tanlab, diagrammani dinamik ravishda yangilash mumkin.
*   **Ma'lumotlar Tooltipi**: Diagrammadagi barlar ustiga sichqonchani olib borganingizda, mamlakat nomi va aniq aholi sonini ko'rsatuvchi ma'lumot tooltipi paydo bo'ladi.
*   **Yagona fayl**: Barcha HTML, CSS va JavaScript kodlari bitta `index.html` faylida jamlangan.
*   **Toza va zamonaviy dizayn**: Loyiha sodda, estetik va foydalanuvchiga qulay interfeysga ega.
