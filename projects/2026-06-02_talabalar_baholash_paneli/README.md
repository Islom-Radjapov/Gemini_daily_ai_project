# student_grades_dashboard

## Project Name: Student Grades Dashboard

## Description
This is a small, fully functional, and beautifully designed dashboard to visualize student academic performance. It provides a quick overview of overall class statistics and a detailed list of individual student grades and statuses. The entire project is self-contained within a single HTML file, with all CSS and JavaScript embedded directly. It uses fake data to demonstrate functionality.

## How to Run
1.  **Save the file:** Save the provided `student_grades_dashboard.html` content into a file named `student_grades_dashboard.html` on your computer.
2.  **Open in browser:** Open the `student_grades_dashboard.html` file using any modern web browser (e.g., Google Chrome, Mozilla Firefox, Microsoft Edge, Apple Safari).
3.  The dashboard will load immediately, displaying student data and statistics.

## Features
*   **Clean and Modern UI:** A visually appealing and intuitive design using modern CSS.
*   **Overall Statistics:** Displays key performance indicators such as total students, overall average grade, highest average grade, and lowest average grade.
*   **Detailed Student List:** Presents a sortable table of students, including their ID, name, major, individual average grade, and academic status (Pass/Fail).
*   **Interactive Search:** Allows users to quickly filter the student list by entering text in the search bar, matching against student names and majors.
*   **Dynamic Data Rendering:** All dashboard elements are dynamically populated using JavaScript from a predefined dataset.
*   **Single File Project:** All HTML, CSS, and JavaScript are encapsulated within one `student_grades_dashboard.html` file, making it easy to deploy and share.
*   **Responsive Design:** Adapts its layout to look good on various screen sizes, from desktops to mobile devices.
*   **No External Dependencies:** Uses only standard web technologies (HTML, CSS, JavaScript) without any external libraries or frameworks.
