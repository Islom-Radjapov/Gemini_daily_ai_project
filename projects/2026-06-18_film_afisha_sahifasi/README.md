# Film Poster Showcase: Interstellar

## Loyiha nomi va tavsifi

"Film Poster Showcase" loyihasi bitta HTML fayl ichida to'liq ishlaydigan, interaktiv film afisha sahifasidir. Ushbu sahifa "Interstellar" filmi haqidagi asosiy ma'lumotlarni (poster, sarlavha, janr, chiqish sanasi, rejissyor, qisqacha mazmun) zamonaviy va jozibali tarzda namoyish etadi. Shuningdek, u "Treylerni tomosha qilish" va "Kuzatuv ro'yxatiga qo'shish" kabi interaktiv tugmalarni o'z ichiga oladi. Loyiha faqat HTML, inline CSS va inline JavaScriptdan foydalanadi, tashqi fayllarga bog'liqliksiz.

## Qanday ishga tushirish mumkinligi

Ushbu loyihani ishga tushirish juda oddiy:

1.  Berilgan kodni `index.html` nomli faylga saqlang.
2.  `index.html` faylini istalgan zamonaviy veb-brauzerda (masalan, Google Chrome, Mozilla Firefox, Safari, Microsoft Edge) oching. Fayl ustiga ikki marta bosing yoki brauzeringizning "Fayl > Faylni ochish" menyusidan foydalaning.

Hech qanday server sozlamalari yoki qo'shimcha o'rnatishlar talab qilinmaydi.

## Xususiyatlar ro'yxati

*   **Bitta Fayl Yechimi:** Barcha HTML, CSS va JavaScript bitta `index.html` faylida birlashtirilgan.
*   **Zamonaviy Dizayn:** Toza, jozibali va zamonaviy UI/UX tamoyillariga rioya qilingan.
*   **Responsive Tartib:** Turli ekran o'lchamlari (mobil, planshet, ish stoli) uchun moslashuvchan dizayn.
*   **Film Ma'lumotlari:** Film nomi, posteri, janri, chiqish sanasi, davomiyligi, reytingi, rejissyori va batafsil qisqacha mazmuni ko'rsatilgan.
*   **Interaktiv "Treylerni tomosha qilish" tugmasi:** Film treylerini yangi brauzer oynasida ochadi.
*   **Interaktiv "Kuzatuv ro'yxatiga qo'shish" tugmasi:** Tugmaning matnini o'zgartiradi va muvaffaqiyatli qo'shilganligi haqida foydalanuvchiga xabar beradi.
*   **Professional Kod:** O'qilishi oson, izohli va yaxshi tashkillashtirilgan kod tuzilishi.
