# tic_tac_toe_game

## Project Name
Tic-Tac-Toe Game (Canvas)

## Project Description
This is a simple, yet fully functional Tic-Tac-Toe game developed using HTML5 Canvas and vanilla JavaScript. The game allows two players to take turns marking spaces on a 3x3 grid, with the goal of getting three of their marks in a row, column, or diagonal. It features an interactive canvas, win/draw detection, and a game reset mechanism, all contained within a single HTML file for ease of use and deployment.

## How to Run
1.  **Save the file:** Copy the content of `tic_tac_toe_game.html` and save it as `tic_tac_toe_game.html` on your local machine.
2.  **Open in Browser:** Navigate to the saved file and open it with any modern web browser (e.g., Chrome, Firefox, Safari, Edge).
3.  **Play:** Click on the cells of the grid to make your moves. Player X starts first.

## Features
*   **Classic 3x3 Grid:** Standard Tic-Tac-Toe board layout.
*   **Interactive Gameplay:** Players can click on empty cells to place their marks ('X' or 'O').
*   **Win Detection:** Automatically detects when a player has won by achieving three in a row (horizontally, vertically, or diagonally).
*   **Draw Detection:** Identifies when the game ends in a draw if all cells are filled without a winner.
*   **Game Status Display:** Shows whose turn it is, and announces the winner or a draw.
*   **Reset Game Functionality:** A dedicated button to clear the board and start a new game at any time.
*   **Single File Deployment:** All HTML, CSS, and JavaScript are embedded within one `tic_tac_toe_game.html` file, making it incredibly easy to share and run.
*   **Clean and Commented Code:** Well-structured and commented JavaScript ensures readability and maintainability.
*   **Modern Design:** Simple, clean, and responsive design with a dark theme.
