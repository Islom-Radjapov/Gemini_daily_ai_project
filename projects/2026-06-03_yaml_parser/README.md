# Simple YAML Parser

## Loyiha nomi va tavsifi (Project Name and Description)

This project delivers a **simple YAML parser** implemented entirely in Python using only standard libraries. It's designed as a practical utility to parse basic YAML configurations and data structures into native Python dictionaries and lists. Unlike more comprehensive YAML libraries (such as `PyYAML`), this parser focuses on handling the most common YAML features, making it lightweight and suitable for educational purposes or environments where external dependencies are strictly prohibited. It provides a foundational understanding of how an indentation-based parser can be built.
