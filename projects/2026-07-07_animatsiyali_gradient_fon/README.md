# animated_gradient_background

## Loyiha nomi: Animatsiyali gradient fon

## Tavsif
Bu kichik loyiha faqat CSS animatsiyalaridan foydalanib, to'liq ekranli, chiroyli va silliq animatsiyali gradient fon yaratadi. Sahifa minimalist dizaynga ega bo'lib, asosiy e'tibor fon animatsiyasiga qaratilgan. Loyiha HTML va CSS-ni bitta faylda birlashtirgan bo'lib, hech qanday tashqi resurs yoki JavaScript talab qilmaydi.

## Qanday ishga tushirish mumkin
Loyihani ishga tushirish juda oddiy:
1.  `index.html` faylini saqlangan joyidan toping.
2.  Faylni ikki marta bosing yoki veb-brauzeringizda (Chrome, Firefox, Safari, Edge va h.k.) oching.
3.  Sahifa darhol yuklanadi va animatsiyali gradient fonni ko'rasiz.

## Xususiyatlar ro'yxati
*   **To'liq CSS animatsiyasi**: Barcha animatsiyalar CSS `background-position` xususiyatini `@keyframes` bilan o'zgartirish orqali amalga oshiriladi.
*   **Minimalist dizayn**: Sahifada oddiy, markazlashtirilgan matn mavjud bo'lib, u animatsiyali fon ustida yaxshi ko'rinadi.
*   **JavaScript yo'q**: Loyiha hech qanday JavaScript kodidan foydalanmaydi.
*   **Bitta fayl**: Barcha HTML va CSS kodlari `index.html` fayli ichida inline joylashgan.
*   **To'liq ekran**: Fon sahifaning butun ko'rish maydonini (viewport) qoplaydi.
*   **Silliq va takrorlanuvchi animatsiya**: Gradient fon silliq va cheksiz tsiklda harakatlanadi.
*   **Zamonaviy va ko'zni qamashtiruvchi ranglar**: Jonli va bir-biriga mos keladigan ranglar palitrasi ishlatilgan.
