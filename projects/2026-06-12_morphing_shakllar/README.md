# Morphing Shapes Animation

## Project Name: Morphing Shapes Animation

## Description:
This project is a small, fully functional web page showcasing a beautiful CSS-only animation of shapes morphing into one another. It demonstrates the power of CSS `border-radius` and `transform` properties combined with `@keyframes` to create a smooth, visually appealing, and dynamic effect without any JavaScript. The design is clean, modern, and professional, focusing on the animation itself.

## How to Run:
1.  **Save the file:** Save the content of `index.html` into a file named `index.html` on your computer.
2.  **Open in browser:** Open the `index.html` file using any modern web browser (Google Chrome, Mozilla Firefox, Microsoft Edge, Safari, etc.).
3.  **Enjoy the animation:** The page will load, and you will immediately see the shape continuously morphing and animating in the center of the screen.

## Features:
*   **Pure CSS Animation:** All morphing effects are achieved exclusively with CSS, using `@keyframes`, `border-radius`, and `transform` properties.
*   **Smooth Transitions:** Uses `ease-in-out` timing functions for fluid and natural-looking shape transformations.
*   **Responsive Design:** The shape maintains its aspect ratio and stays centered, adapting well to different screen sizes.
*   **Minimalist Aesthetic:** A clean, uncluttered design with a focus on the animated element.
*   **Continuous Loop:** The animation runs infinitely, providing a perpetual visual display.
*   **No JavaScript:** Absolutely no JavaScript is used, keeping the project lightweight and performant.
